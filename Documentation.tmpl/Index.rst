Additional Information
=============================================================

.. only:: html

	:Classification:
		additional_information

	:Version:
		|release|

	:Language:
		en

	:Description:
		The extension provides additional information on a tt_content element in the backend. For backend editors it can be very annoying to open up eacy tt_content element to show the settings concerning space before or after as well as the layout. This extension shows an additional collapsible panel in the backend attached to each element. It can be collapsed to show these informations, without reloading the page properties page.
